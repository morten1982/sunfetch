# sunfetch
Weather for the Terminal


![alt text](https://github.com/morten1982/sunfetch/blob/main/sunfetch.png)


# Description
The current Weather and Forecast for the Terminal -> using the OpenWeatherMap API


# Requirements
   -> Enter your OpenWeatherMap API Key in the source code in the "const" section and
   -> compile it using: -> fpc sunfetch.pas
